With this plugin, an notification can be added to old posts.

== Description ==

With Old Post Notifier, you can add a warning to your posts published before a while.
Warning message and warning template can be changed optionally.

== Installation ==

You can install this plugin

1. Upload `old-post-notifier` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After the plugin is activated, you can make adjustments from `Settings - Old Post Notifier`

== Frequently Asked Questions ==

= How can I arrange the day in the plugin? = 

From `Settings - Old Post Notifier` , adjust the number of days dating from you want to add the message, to the `Minimum Days` part.
